# 飞机大战（EasyX / VS2022）

## 开发环境

- Visual Studio 2022，平台工具集 `v143`
- EasyX `20220901`（低于 2023 的版本）
- C++20；不依赖其他游戏框架

代码只使用 EasyX 2022 已提供的接口，包括 `IMAGE`、`ExMessage`、批量绘图、遮罩贴图、`SetWorkingImage` 和 `setorigin`。

## 打开和运行

1. 为 VS2022 安装 EasyX 20220901。
2. 双击根目录下的 `飞机大战.sln`（推荐），也可打开 `飞机大战.slnx` 或 `飞机大战\飞机大战.vcxproj`。
3. 选择 `Debug | x64` 或 `Release | x64`，启动“飞机大战”。

图片、排行榜和存档使用相对路径，VS 调试工作目录应保持为项目目录。

## 音乐分配

- 主菜单、结算和暂停界面：`menu_loop.mp3`
- 第一、二关普通阶段：`stage12_loop.mp3`
- 第一、二关 Boss：`boss12_loop.mp3`
- 第三关（含 Boss）：`stage3_loop.mp3`
- 无尽模式：`endless_loop.mp3`
- 玩家与僚机射击：`shoot_sfx.wav`
- 飞机爆炸：`explosion_sfx.wav`

原始 OGG 文件保留在 `music` 中；运行时使用转换后的 MP3/WAV，通过 Windows 自带 WinMM 播放，不增加游戏运行依赖。

## 操作

- P1：`W/A/S/D` 或鼠标拖动；空格射击；`B` 使用炸弹。
- P2：方向键移动；`J`、主键盘 `1` 或小键盘 `1` 射击；`N` 使用炸弹。
- 双人模式中，B/N 在 280ms 内同时按下会触发更强的合体炸弹，炸弹数量共享。
- `Esc`：暂停、保存或返回菜单。

## 已实现玩法

- 关卡模式和无尽模式，以及互相独立的排行榜。
- 三关 Boss：第一关移动散射教学，第二关召唤双编队夹击，第三关环形弹、预警激光和低血量狂暴。
- 所有普通敌机血条、Boss 血条受击闪白、命中闪烁、击杀震屏、70ms 爆炸帧、事件音效和炸弹冲击波。
- 连击倍率、擦弹得分、关末随机三选一升级；升级在本次挑战内有效，游戏结束后重置。
- 火力、无人机、生命、炸弹、护盾、磁吸、冻结、穿透、过载和双人复活共十类道具。
- 双人坚持复活、近距离联结护盾、独立击杀/贡献统计；道具归拾取者，炸弹共享。
- 敌机出现后立即开始射击；追踪敌机只在 X 轴追踪，在 Y 轴持续下落。
