# 飞机大战（EasyX / Visual Studio）

一款以“小猫敌机”为主题的纵向射击游戏，支持单人、双人合作、三关挑战和无尽模式。游戏内主菜单现已提供“游戏说明”入口。

## 开发与运行环境

- Visual Studio 2022 或更新版本
- EasyX 图形库
- C++20，Windows x64

打开根目录下的 `飞机大战.sln`，选择 `Debug | x64` 或 `Release | x64` 后运行。图片、音乐、排行榜和存档均使用相对路径，因此运行时工作目录应为 `飞机大战` 项目目录。

## 基础操作

### 玩家 1

- 移动：`W / A / S / D`
- 射击：`空格`
- 使用炸弹：`B`

### 玩家 2

- 移动：方向键
- 射击：`J`、主键盘 `1` 或小键盘 `1`
- 使用炸弹：`N`

### 通用操作

- `Esc`：打开暂停菜单，可继续游戏、保存并返回菜单、直接返回菜单或退出游戏
- 双人模式下，`B` 与 `N` 在 280 毫秒内按下会触发威力更强的合体炸弹

## 游戏模式

- 关卡模式：开始前可选择简单或困难难度，依次挑战三关。困难难度保留原有敌机数量、刷新节奏和首领强度；简单难度为新手通关模式，普通敌机整体压力约为困难的四分之一，开局自带 2 枚炸弹且道具掉落率更高。简单难度首领为 50% 生命，弹幕间隔加倍、密度和速度降低；第二关首领不再召唤增援，第三关最多只保留 1 只低频增援。首领不显示普通的阶段切换文字，仅在进入狂暴前显示短暂红色预警。首领分别在本关进行约 60、75、90 秒后登场。前两关结束后可三选一升级，第三关结束后直接显示通关得分。
- 闯关第 1 关沿用原背景，第 2、3 关使用各自独立的纵向滚动背景。
- 无尽模式：敌人数量、刷新速度与强度会随时间持续提升，目标是尽可能获得高分。
- 两种模式使用独立排行榜。

## 战斗与得分

- 击落敌机可获得分数；连续击落会累计连击并提高分数倍率。连击期间，右上角只显示当前倍率，连击中断后自动隐藏。
- 贴近敌弹并成功闪避可触发“擦弹”，获得额外分数。
- 普通敌机统一使用小猫形象，并具有血条、受击反馈和爆炸效果；首领会根据关卡和血量切换攻击阶段。
- 最终首领战会持续召唤能够射击的小猫护卫。
- 炸弹可以清除敌弹并对全场敌人造成伤害，对首领同样有效。

## 道具

- 火力：暂时增强射击
- 僚机：召唤跟随射击的小飞船；最后 3 秒会从慢闪逐渐变为快闪
- 生命：恢复生命值
- 炸弹：增加炸弹数量
- 护盾：抵挡一次伤害，可通过升级提高容量
- 磁吸：自动吸引附近道具，并显示从外侧向飞船中心收缩的吸入粒子；最后 3 秒加速闪烁
- 冻结：只减缓敌机与敌弹，玩家速度不变；结冰效果仅显示在屏幕四周，不遮挡中央战斗区域；最后 3 秒加速闪烁
- 穿透：子弹可继续穿过目标
- 过载：短时间获得高射速与五路火力，结束后进入短暂冷却；由于清场能力很强，它是掉落概率最低的道具
- 复活：仅在双人模式掉落，可立即复活队友；队友存活时转为治疗拾取者

## 双人合作机制

- 两架飞船靠近时会建立联结护盾，周期性抵挡伤害。
- 一名玩家阵亡后，只要队友继续坚持 8 秒，阵亡玩家便会自动复活。
- 炸弹数量由两名玩家共享，道具效果归拾取者所有。
- 界面会分别显示两名玩家的击杀和贡献。

## 音乐与资源

- 菜单、关卡、首领战、无尽模式、暂停和结算界面使用不同背景音乐。暂停音乐在启动时预载，暂停期间不会消耗关卡时间、限时道具或冷却时间。
- 射击、爆炸、玩家死亡和小猫叫声使用独立音效。
- 游戏素材位于 `飞机大战/images` 和 `飞机大战/music`。

## 存档说明

游戏中按 `Esc`，选择“保存并返回菜单”即可保存当前玩家、模式、关卡、分数、生命、护盾、道具剩余时间、升级状态和首领状态。主菜单的“继续存档”可恢复进度。
